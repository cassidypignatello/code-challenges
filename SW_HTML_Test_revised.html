  • Q: Describe the differences between the IE6, IE7, and W3C box models.

    In the IE6/IE7 box model, the width of an element is measured between the borders of the box, including both the padding and the border. In the W3C box model, the width of an element is measured by the width of the content of the box, excluding padding and border. In CSS3, the W3C included a box-sizing property which allows you to switch between box models. Using the declaration box-sizing: border-box will switch to the IE6/IE7 model, while box-sizing: content-box will use the W3C model. Some people find the W3C model counter-intuitive, and usually would make the argument that logically the width of a physical box would be measured from border to border, and not by it's content. Ultimately, it's a subjective argument, and enforcing standards compliance is more important than personal preference.
